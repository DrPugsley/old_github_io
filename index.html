Welcome to Tuxcoin!

Supporting the OSS Community


        Tuxcoin is a cryptocurrency intended to support Open-Source Software projects. Tuxcoin is Proof-of-Work, utilizing the low-power consuming Allium algorithm. Support of OSS projects comes in the form of donations to projects that the community chooses and votes on. The project teams that receive donations are free to do with the cryptocurrency as they will. Our donations will further encourage OSS growth and adoption, which is a key component to our digital future.
The Tuxcoin Development Team will set in motion a plan to donate to 5 projects on a monthly basis, all supported by the community.

Getting Started

How to Set Up a Wallet

To get your wallet, simply click on the image that corresponds to your operating system

For more options (such as cli releases and releases for architectures other than x86_64), you can visit the releases page on our Github repo.

How to Mine



In order to mine Tuxcoin you must first set up a wallet. Please visit the link below to get the most recent version of the wallet for your operating system:
https://github.com/bleach86/tuxcoin-v2/releases

Once you have the wallet installed, click the File Menu and then Releases to get your first wallet address.
Note: Keep your wallet protected! Never give out your private keys. It is also highly recommended you back up the wallet.dat file you’ll find in your home directory.

Now you’re ready to start mining! Mining Tuxcoin can be accomplished with either a GPU or CPU. The mining software for either can be found here:
GPU: https://github.com/tpruvot/ccminer/releases
CPU: https://github.com/JayDDee/cpuminer-opt/releases

Each miner release comes with a few example batch files. Please either modify one you find there or create your own. Here is an example for use with ccminer:

:start
ccminer-x64 -a allium -o stratum+tcp://POOLADDRESS:PORT -u WALLETADDRESS
goto start
pause

Be sure to replace POOLADDRESS:PORT with the connection details you find at the pool’s site. Also, WALLETADDRESS should be replaced with the address you found in your wallet.
Note: This example script is for pools running with the NOMP software. Other pools have different connection strings. Please cater your batch files to what the pool is expecting – each pool should provide these details to you.

How to Set Up a Pool



Every decentralized cryptocurrency relies on balanced workloads to ensure there is no abuse of the blockchain. We achieve this through multiple sources of mining – the biggest of which is pools.

Please bear in mind: setting up a pool is a considerable undertaking. If you haven’t set one up before you may find some of the process to be rather daunting. We encourage you to try and if you are stuck, there are plenty of helpful people on our Discord!

There are three major pool software options. Please follow the links below to research the one that best fits your needs:

https://github.com/MPOS/php-mpos

https://github.com/bleach86/tux-nomp

https://github.com/tpruvot/yiimp

Install instructions for tux-nomp can be found here

https://github.com/bleach86/tux-nomp/blob/master/docs/README.md
